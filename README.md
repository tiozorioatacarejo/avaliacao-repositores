# Avaliação de Performance de Repositores

Aplicação web simples para auditoria e avaliação de performance de repositores.

## Funcionalidades

- Cadastro de data, repositor, setor e auditor.
- Avaliação dos pilares:
  - Ruptura
  - Validade/FIFO
  - Precificação
  - Organização e limpeza
  - Execução de ofertas
- Cálculo automático das notas por pilar, resultados ponderados, nota final e classificação.
- Exportação dos dados para planilha Excel `.xlsx`.

## Como usar

Abra o arquivo `index.html` em um navegador.

Para publicar no GitHub Pages, envie os arquivos deste projeto para um repositório e ative o Pages nas configurações do repositório.
